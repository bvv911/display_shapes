### Display shapes
Пример программы для вывода геометрических фигур на экран (треугольник, круг, квадрат, прямоугольник) на C++17. Для вывода графики на экран используется библиотека [SFML](https://www.sfml-dev.org). 
### Сборка и запуск
Библиотека SFML скачивается и собирается при помощи команды FetchContent в CMake. Для ее компиляции вероятно потребуется установить дополнительные зависимости, подробнее см. на странице документации проекта [ccылка](https://www.sfml-dev.org/tutorials/3.0/getting-started/cmake/)
Для отрисовки фигур в качестве аргументов передается название фигуры за которым следует число этих фигур.
Пример: `./main triangle 3 circle 2 rectangle 1 square 2`
